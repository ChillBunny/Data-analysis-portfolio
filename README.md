# Data Analysis Portfolio — Aglis Tamayo

## 🇬🇧 English version

Data analyst from Santiago de los Caballeros, Dominican Republic. Python, SQL, statistics, and a special interest in financial markets.

Each folder is a self-contained project: code, a **bilingual README (English + Español)** and an `images/` folder with the resulting charts.

Projects in order of depth and interest:

| # | Project | Question | Stack |
|---|---------|----------|-------|
| 1 | [fifa-hypothesis-test](./fifa-hypothesis-test) | Are more goals scored in women's FIFA World Cup matches than in men's? A formal hypothesis test. | Python · Pandas · Pingouin · SciPy |
| 2 | [nobel-prize-winners](./nobel-prize-winners) | Who wins the Nobel Prize, and how has female representation evolved since 1901? | Python · Pandas · NumPy · Seaborn |
| 3 | [la-crime-patterns](./la-crime-patterns) | When, where and against whom do crimes happen in Los Angeles? 185k real records, population-weighted. | Python · Pandas · Seaborn |
| 4 | [nyc-schools-scores](./nyc-schools-scores) | Which NYC public schools lead the SAT, and which borough varies the most? | Python · Pandas · Seaborn |
| 5 | [netflix-eda](./netflix-eda) | What does the Netflix catalog look like across decades, genres and durations? | Python · Pandas · Seaborn |
| 6 | [broker-kpi-analysis](./broker-kpi-analysis) | How would you compare brokers in the Dominican securities market? A KPI framework demo **(simulated data)** | Python · Pandas · Matplotlib · Seaborn |

**Note on the broker project:** its data is SIMULATED. Broker names and SIVCV codes belong to real registered brokers of the Dominican securities market (public information), but every figure is fictional — the project is about the comparative KPI framework, not the numbers. See its full disclaimer.

The FIFA and Netflix datasets are included so those projects run end to end; the remaining datasets come from DataCamp guided projects.

## 🇪🇸 Versión en español

Analista de datos de Santiago de los Caballeros, República Dominicana. Python, SQL, estadística, y un interés especial en los mercados financieros.

Cada carpeta es un proyecto autocontenido: código, un **README bilingüe (inglés + español)** y una carpeta `images/` con los gráficos resultantes.

Proyectos en orden de profundidad e interés:

| # | Proyecto | Pregunta | Stack |
|---|----------|----------|-------|
| 1 | [fifa-hypothesis-test](./fifa-hypothesis-test) | ¿Se anotan más goles en los mundiales femeninos que en los masculinos? Un test de hipótesis formal. | Python · Pandas · Pingouin · SciPy |
| 2 | [nobel-prize-winners](./nobel-prize-winners) | ¿Quién gana el Premio Nobel, y cómo ha evolucionado la representación femenina desde 1901? | Python · Pandas · NumPy · Seaborn |
| 3 | [la-crime-patterns](./la-crime-patterns) | ¿Cuándo, dónde y contra quién ocurren los crímenes en Los Ángeles? 185 mil registros reales, ponderados por población. | Python · Pandas · Seaborn |
| 4 | [nyc-schools-scores](./nyc-schools-scores) | ¿Qué escuelas públicas de NYC lideran el SAT, y qué distrito varía más? | Python · Pandas · Seaborn |
| 5 | [netflix-eda](./netflix-eda) | ¿Cómo luce el catálogo de Netflix a través de décadas, géneros y duraciones? | Python · Pandas · Seaborn |
| 6 | [broker-kpi-analysis](./broker-kpi-analysis) | ¿Cómo compararías a los corredores del mercado de valores dominicano? Una demo de marco de KPIs **(datos simulados)** | Python · Pandas · Matplotlib · Seaborn |

**Nota sobre el proyecto de corredores:** sus datos son SIMULADOS. Los nombres y códigos SIVCV pertenecen a corredores reales registrados del mercado de valores dominicano (información pública), pero todas las cifras son ficticias — el proyecto trata del marco comparativo de KPIs, no de los números. Ver su descargo de responsabilidad completo.

Los datasets de FIFA y Netflix están incluidos para que esos proyectos corran de punta a punta; los demás provienen de proyectos guiados de DataCamp.
